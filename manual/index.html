1. はじめに — アカウント作成

Relatistaを使い始めるには、まずアカウントを作成します。

  1. サインアップページにアクセスします
  2. メールアドレスとパスワードを入力して「登録」をタップします
  3. 確認メールが届いたら、リンクをクリックして認証を完了します
  4. オンボーディング画面で、以下の情報を入力します:
    • ニックネーム（アプリ内で表示される名前）
    • 生年月日（年・月は必須、日は任意）
    • 性別・国（任意）

Googleアカウントでの登録も可能です。サインアップページの「Googleで続ける」ボタンをタップしてください。

2. ホーム画面の使い方

ログインすると、ホーム画面に今日のガイダンスが表示されます。

日次ガイダンス

毎日、あなたの状況に合わせたパーソナライズされたアドバイスが届きます。ガイダンスは朝に更新されます。

天気の選択

ホーム画面上部の天気アイコンをタップすると、今日の天気を手動で選択できます。天気情報はガイダンスの内容に反映されます。

週次・月次タブ

画面上部のタブを切り替えると、より長いスパンのガイダンスを確認できます。

  • 週次: 今週の傾向とアドバイス
  • 月次: 今月の全体的なテーマ

3. 人物の登録

大切な人を登録して、相性分析やAI相談を活用しましょう。

  1. 画面下部のナビゲーションから「人物」タブをタップします
  2. 「人物を登録」ボタンをタップします
  3. 以下の情報を入力します:
    • ニックネーム（例: たかしさん、上司A など）
    • 生年月日
    • 関係性（上司・同僚・友人・家族・パートナーなど）
    • 性別・国（任意）
  4. 「登録」をタップして完了です

相性スコアの見方

人物を登録すると、自動的に相性スコアが計算されます。スコアは0〜100の数値で表示され、関係性の傾向がわかります。

クイック分析を実行すると、より詳しい関係性の分析結果を確認できます。

4. AIに相談する

人間関係の悩みをAIに相談して、具体的なアドバイスをもらいましょう。

  1. 相談したい人物のページを開きます
  2. 「相談する」ボタンをタップします
  3. 相談内容を入力します（例: 「最近コミュニケーションがうまくいかない」）
  4. AIが分析結果をもとに、具体的なアドバイスを返します

相談内容は履歴として保存されるため、後から振り返ることができます。

5. ホーム画面に追加（PWA）

Relatistaはウェブアプリ（PWA）です。ホーム画面に追加することで、通常のアプリと同じように使えます。

iPhoneの場合

  1. Safariで app.relatista.com を開きます
  2. 画面下部の共有ボタン（四角に上矢印のアイコン）をタップします
  3. 「ホーム画面に追加」を選択します
  4. 「追加」をタップして完了です

Androidの場合

  1. Chromeで app.relatista.com を開きます
  2. 右上のメニュー（三点アイコン）をタップします
  3. 「ホーム画面に追加」または「アプリをインストール」を選択します
  4. 「追加」をタップして完了です

6. 通知の設定

プッシュ通知を有効にすると、毎日のガイダンスが届いたときにお知らせを受け取れます。

  1. アプリの設定画面を開きます
  2. 「通知」のトグルをONにします
  3. ブラウザの通知許可ダイアログが表示されたら「許可」をタップします

通知はいつでも設定画面からOFFにできます。ブラウザの設定からも通知を管理できます。
